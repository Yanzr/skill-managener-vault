# skill-managener-vault

这是一个简要的 skills 分类索引，方便快速查找可用能力。

> 说明：`nature-*` 系列是一组 Nature 风格学术工作流技能；这里按可直接使用的技能名做简要分类。

## 学术写作与论文工作流

- `deeppapernote`：论文精读、笔记整理与结构化提炼。
- `nature-academic-search`：多源学术检索与引用核验。
- `nature-citation`：Nature 风格引文与参考文献工作流。
- `nature-data`：学术数据可用性、元数据与声明处理。
- `nature-figure`：Nature 风格论文图表与配图制作。
- `nature-paper-to-patent`：论文到中文专利的起草与转写。
- `nature-paper2ppt`：论文到汇报 PPT 的提炼与排版。
- `nature-polishing`：Nature 风格学术润色与措辞优化。
- `nature-reader`：整篇论文的 Markdown 读取与结构化提取。
- `nature-response`：审稿意见逐点回复与 rebuttal 起草。
- `nature-reviewer`：模拟 Nature 审稿人视角做预投稿评审。
- `nature-writing`：Nature 风格论文写作与章节起草。

## 图像、设计与可视化

- `ccf-figure`：将 AI/CS 论文内容转成顶会级科研配图。
- `scientific-figure-generator`：通用科研图生成与提示词辅助。
- `ian-xiaohei-illustrations`：偏展示/信息图风格的中文插画。
- `universal-image`：通用图片生成与调参辅助。
- `canvas-design`：视觉设计理念与画面风格构建。
- `frontend-design`：前端界面视觉设计建议。
- `frontend-slides`：零依赖、动画化 HTML 演示稿。
- `screenshot`：网页截图与页面捕获辅助。

## 文档、办公与文件处理

- `document-reader`：文档阅读与内容提取。
- `doc-coauthoring`：协作式文档共创流程。
- `docx`：DOCX 文档创建、编辑与分析。
- `pdf`：PDF 处理与转换指导。
- `pptx`：PPTX 生成与编辑指导。
- `xlsx`：表格 / Excel 处理与输出规范。
- `planning-with-files-zh`：基于文件的长期规划与任务拆解。

- `git-commit`：按 Conventional Commits 规范生成提交信息。
- `changelog-generator`：将技术提交整理成面向用户的更新日志。
- `find-skills`：搜索、发现并安装 skills。
- `find-skills-1.0.0`：`find-skills` 的旧版本 / 打包版本。

- `openclaw-medical-skills`：OpenClaw 医疗技能库的 Codex 入口。
